# Loto Análise Pro

Aplicativo Android para análise estatística das loterias brasileiras.

## Objetivos

- Atualização automática dos resultados
- Cache offline
- Estatísticas avançadas
- Gerador inteligente de jogos
- Conferidor de apostas
- Dashboard moderno
- Material Design 3
- Kotlin + Jetpack Compose

## Tecnologias

- Kotlin
- Jetpack Compose
- Material Design 3
- MVVM
- Clean Architecture
- Hilt
- Room
- Retrofit
- WorkManager
- DataStore
